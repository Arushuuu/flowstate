#  FlowState - Menstrual Cycle Tracker

A secure, full-stack application designed to track menstrual health

##  Features
- **Secure Auth:** Password encryption using Bcrypt.
- **Persistent Data:** MongoDB integration for user accounts.
- **Cycle Analytics:** Calculates Next Period, Follicular, Ovulation, and Luteal phases.
- **Adaptive UI:** Responsive design using CSS Flexbox/Grid.

##  Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Security:** Bcrypt.js, CORS

##  Setup
1. Clone the repo: `git clone <your-url>`
2. Install: `npm install`
3. Start: `node server.js`
